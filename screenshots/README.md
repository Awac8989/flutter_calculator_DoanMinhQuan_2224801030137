# Screenshots

This directory contains screenshots of the calculator app demonstrating various features:

## Required Screenshots:

### Core Modes
- `basic_mode.png` - Basic calculator interface
- `scientific_mode.png` - Scientific calculator with advanced functions  
- `programmer_mode.png` - Programmer mode with base conversion

### Bonus Features
- `voice_input.png` - Voice input feature demonstration
- `graph_plotting.png` - Mathematical function graphing
- `export_features.png` - CSV/PDF export options

### UI Features
- `dark_mode.png` - Dark theme interface
- `landscape_mode.png` - Landscape orientation layout

## How to Take Screenshots:
1. Run the app: `flutter run`
2. Navigate to each mode/feature
3. Take screenshots using emulator or device
4. Save with appropriate naming convention
5. Update README.md with actual screenshot links

## Screenshot Specifications:
- Format: PNG
- Resolution: At least 720x1280 (mobile)
- Quality: High quality, clear interface elements
- Orientation: Both portrait and landscape where applicable
